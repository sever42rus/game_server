# Логирование игрового цикла

Логирование агрегированных метрик `_world_loop` в `src/infrastructure/server.py` удалено.

Удалены:
- счетчики и накопители метрик игрового цикла;
- вызов `_print_world_loop_metrics`;
- сбор метрик сна между тиками;
- метод `_print_world_loop_metrics`.

Игровой цикл продолжает обрабатывать команды, выполнять тик игры и публиковать исходящие события.
