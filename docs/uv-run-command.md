# Запуск сервера через uv

Для удобного локального запуска в проекте зарегистрирован консольный скрипт
`game-server`.

## Команды

Первичная синхронизация окружения:

```bash
uv sync
```

Запуск сервера:

```bash
uv run game-server --host 127.0.0.1 --port 9000
```

## Техническая деталь

Команда `game-server` указывает на функцию `run` из `src.main`. Эта обертка
нужна, потому что `project.scripts` ожидает обычную синхронную функцию, а
основной запуск сервера построен вокруг `asyncio`.

Аргументы `--host` и `--port` обязательны. `--host` принимает только
корректный IPv4 или IPv6 адрес, а `--port` валидируется как TCP-порт в
диапазоне `1..65535`.
